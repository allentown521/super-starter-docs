ShipThing 使用 [Ultracite](https://ultracite.dev) 进行代码格式化和代码检查。Ultracite 是基于 [Biome](https://biomejs.dev) 的预配置设置，Biome 是一个高性能的 Rust 工具链，包含格式化器、代码检查器等，支持 JavaScript 和 TypeScript。

## 优势

Ultracite 提供了以下优势：

- <Icon icon="check" iconType="solid" /> 零配置 - 开箱即用
- <Icon icon="check" iconType="solid" /> 极快的性能
- <Icon icon="check" iconType="solid" /> 整个项目代码风格一致
- <Icon icon="check" iconType="solid" /> 内置代码检查规则
- <Icon icon="check" iconType="solid" /> 支持 TypeScript

## 使用

格式化器和代码检查器已在您的项目中自动配置，并会在保存时运行。如果您想手动运行它们，可以使用以下命令：

- `pnpm lint` - 检查所有应用和包中的文件
- `pnpm format` - 检查并修复所有应用和包中的文件
