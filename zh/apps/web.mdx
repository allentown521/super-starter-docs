SuperStarter自带一个默认的网站应用，通常称为Landing Page，位于`apps/web`文件夹中。

## 概述

它包含了启动页面所需的一切功能，我们使用[blocks组件](/zh/packages/design-system/blocks)构建，您可以通过修改[配置](/zh/setup/configuration/app-config)在几分钟内像搭积木一样构建您的网站。

<Tip>
  您知道吗？我们提供了数十种区块，包括`header`、`footer`、`hero`、`features`、`subscribe`、`waitlist`、`faq`、`pricing`、`testimonials`、`cta`、`usage`、`login`、`brand`和`contact`。您可以在[blocks组件](/zh/packages/design-system/blocks)中找到它们。
  
  您可以在[storybook](/zh/apps/storybook)中查看它们的使用示例。
</Tip>

## 功能特性

- **设计系统**：应用连接了[设计系统](/zh/packages/design-system)，包含各种组件、钩子和工具，帮助您快速开始
- **内容管理系统**：应用连接了[CMS](/zh/packages/cms)包，为您的类型安全博客、文档和法律页面提供支持
- **SEO优化**：应用连接了[SEO](/zh/packages/seo)包，优化网站搜索引擎表现
- **分析统计**：应用连接了[Analytics](/zh/packages/analytics)包，追踪访客行为
- **可观测性**：应用连接了[Observability](/zh/packages/observability)包，追踪错误和性能指标

## 开发

要启动应用，请在根目录中运行以下命令：

```sh Terminal
pnpm --filter web dev
```
