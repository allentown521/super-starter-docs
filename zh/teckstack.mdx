### Turborepo

[Turborepo](https://turbo.build/repo) 是一个 Monorepo 工具，可帮助您管理项目的依赖关系和脚本。我们选择 Monorepo 架构来更轻松地管理不同功能模块，并实现各包之间的代码共享。该架构也便于您后续添加更多应用（如移动应用或其他前端）并复用现有代码库。

## Next.js

[Next.js](https://nextjs.org/) 是最流行的 React 框架之一。我们将其用于 SuperStarter 的前端开发，并利用其路由处理器来提供应用程序的 API。

## Prisma

[Prisma](https://www.prisma.io/) 是一个 ORM（对象关系映射）工具，可帮助您以类型安全的方式访问数据库。我们使用 Prisma 定义数据库模式并生成应用程序所需的 TypeScript 类型。它也帮助我们以类型安全的方式编写查询，并轻松迁移数据库模式。

我们默认使用 [PostgreSQL](https://www.postgresql.org/)，但您可以通过更新少量配置轻松切换至 MySQL、SQLite 或其他支持的数据库。

## Hono

[Hono](https://hono.dev/) 是一个极简的 Web 服务器框架，用于构建 API。它使您能够轻松定义路由和处理请求响应。可运行在边缘计算、无服务器函数或传统长时运行服务器上，因此适用于任何环境和提供商。其附带的 RPC 客户端允许您从前端以类型安全的方式直接调用自身函数。

## Tailwind CSS & Radix UI

[Tailwind CSS](https://tailwindcss.com/) 是一个实用优先的 CSS 框架，无需编写 CSS 即可构建自定义设计。我们还使用 [Radix UI](https://www.radix-ui.com/) 作为无头组件库，使用 [shadcn/ui](https://ui.shadcn.com/) 作为设计系统，通过单一命令即可生成预设样式的组件。

## Clerk

[Clerk](https://clerk.com/) 是用户管理服务，可轻松为应用添加身份验证功能。它提供用户管理仪表板、用户管理 API 和前端 SDK。我们用它来管理用户并进行身份验证。

## Fumadocs

[Fumadocs](https://fumadocs.com/) 是基于 Markdown 的内容管理系统，帮助您在代码库中结构化地管理内容。我们用它通过 MDX 来管理博客、文档和其他内容页面（如法律条款页）。
